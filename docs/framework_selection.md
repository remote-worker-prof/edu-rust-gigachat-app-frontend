# Выбор фреймворка Web UI на Rust (учебная версия)

Документ объясняет, почему в учебном курсе выбран именно Yew, и кратко
поясняет, что такое фреймворк.

## 1. Что такое фреймворк

**Фреймворк** — это набор готовых инструментов и правил, которые помогают
строить приложение быстрее и более упорядоченно. Он задаёт структуру
проекта и типовые способы решения задач.

## 2. Критерии выбора для учебного курса

Для учебных проектов важны:

- стабильность и предсказуемость;
- понятная документация;
- большое сообщество (много примеров и ответов на вопросы);
- невысокий порог входа.

## 3. Итоговый выбор

**Yew** выбран как основной фреймворк для UI.

### Почему именно Yew

- зрелая и стабильная экосистема;
- понятная модель компонентов;
- хорошие учебные материалы;
- подходит для демонстрации базовых принципов web‑разработки.

## 4. Альтернативы (для справки)

- **Dioxus** — активно развивается, позволяет собирать UI для разных платформ.
- **Leptos** — современный подход с возможностью серверной отрисовки (SSR).

> Примечание: альтернативы изучаются только по отдельному решению
> преподавателя. Базовый курс ориентирован на Yew.
