# Учебные материалы по лабораторным работам (UI)

Данная директория содержит материалы для лабораторных работ по учебному
веб‑приложению `rust-gigachat-webapp` (клиент к API `rust-gigachat-app`).

## Содержание

- `lab_2/` — Лабораторная работа №2: подключение UI к API и устранение CORS.

## Рекомендуемый порядок работы

1. Ознакомиться с основной документацией проекта:
   - `docs/build_and_run.md` — запуск и сборка UI;
   - `docs/api_contract.md` — контракт API;
   - `docs/common_issues.md` — частые проблемы и их причины;
   - `docs/ui_ux_requirements.md` — требования к UI/UX;
   - `docs/stack_guide.md` — обзор используемого стека.
2. Перейти в нужную лабораторную работу и следовать инструкциям.

Документация написана для начинающих и содержит пояснения терминов и шагов.
