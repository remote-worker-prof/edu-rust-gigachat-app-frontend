# Частые проблемы (веб-приложение)

## API недоступно

Причины:
- Бэкенд не запущен.
- Неверный API_BASE_URL.
- Сеть или блокировка порта.

Решение:
- Проверить /health на бэкенде.
- Уточнить адрес и порт.

## Ошибка CORS

Причина:
- Браузер блокирует запросы между разными доменами.

Решение:
- Включить CORS на стороне бэкенда или использовать прокси.

## Пустой ответ на запрос

Причина:
- Вопрос пустой или невалидный JSON.

Решение:
- Проверить валидацию на клиенте.

## Предупреждения от зависимостей

Причина:
- Транзитивные зависимости могут давать предупреждения компилятора.

Решение для учебного проекта:
- Фиксировать как известное ограничение.
- Не применять форки без отдельного методического решения.

## Trunk падает с `invalid value '1' for '--no-color'`

Причина:
- В окружении задано `NO_COLOR=1`, а Trunk ожидает `true/false`.

Решение:
- Запускайте Trunk так: `NO_COLOR=true trunk build` или `NO_COLOR=true trunk serve`.
