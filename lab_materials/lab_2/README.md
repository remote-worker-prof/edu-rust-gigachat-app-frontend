# Лабораторная работа №2 (UI)

Тема: подключение UI к API и устранение CORS.

## Содержание

1. **lab_work.md** — основной текст лабораторной работы (теория + шаги).
2. **group_work_guide.md** — методические указания по групповой работе.
3. **library_reference.md** — справочник по библиотекам UI‑проекта.
4. **resources_collection.md** — подборка полезных ссылок и материалов.

## Для преподавателя

Рекомендуется начать с `lab_work.md`, затем использовать справочник и
ресурсную подборку для дополнительных материалов.
